## Tech Stack

**Client:** React, Vite, SCSS, Yarn, Typescript

**Version Control:** Github

**Deployment:** Vercel


## Run Locally

Clone the project

```bash
  git clone https://github.com/Sauravionic/StarChef-2-Chefkart.git
```

Go to the project directory

```bash
  cd frontend
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn run dev
```
